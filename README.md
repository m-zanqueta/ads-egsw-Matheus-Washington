# Curupira - Sistema de Monitoramento de Compostagem

## Sobre o Projeto
O Curupira é uma solução inovadora desenvolvida pelo Grupo RAIZ para monitoramento inteligente de compostagem. Este site institucional apresenta a tecnologia Curupira ao mercado, destacando seus benefícios e funcionalidades para revolucionar o processo de compostagem.

## Tecnologias Utilizadas
- HTML5, CSS3 e JavaScript para desenvolvimento front-end
- Git e GitHub para versionamento e colaboração
- Metodologia ágil com Git Flow adaptado

## Funcionalidades Principais
- Página inicial com apresentação da solução
- Seção "Sobre" com história e valores da empresa
- Página de contato com formulário funcional
- Galeria com sistema de filtros para imagens do produto
- Design totalmente responsivo para todos os dispositivos
- Componentes modulares reutilizáveis (header e footer)
