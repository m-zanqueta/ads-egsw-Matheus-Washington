# Changelog

Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/spec/v2.0.0.html).

## [1.3.0] - 2025-11-10

### Adicionado
- Página de Galeria com sistema de filtros
- CSS específico para a galeria (`galeria.css`)
- JavaScript para funcionalidades de filtragem (`galeria.js`)
- Layout responsivo para a galeria de imagens

### Corrigido
- Problema de desconfiguração do header no index
- Compatibilidade entre CSS do header e páginas específicas
- Responsividade do header em dispositivos móveis

## [1.2.0] - 2025-11-09

### Adicionado
- Página "Sobre" com história e valores do projeto Curupira
- CSS específico para a página sobre (`sobre.css`)
- Seções: História, Filosofia, Equipe e Valores
- Design ambientalizado com cores verdes sustentáveis

### Modificado
- Atualização da paleta de cores para tema de compostagem
- Melhoria na consistência visual entre páginas
- Header e footer ambientalizados mantendo fundo claro

## [1.1.0] - 2025-11-09

### Adicionado
- Página de Contato com formulário funcional
- CSS específico para contato (`contato.css`)
- Seção hero com imagem de fundo
- Informações de contato e formulário side-by-side

### Modificado
- Separação dos CSS em header, footer e body
- Estrutura modularizada para reutilização
- Melhoria na organização dos arquivos

## [1.0.0] - 2025-11-08

### Adicionado
- Versão inicial do projeto Curupira
- Estrutura base do site institucional
- Página inicial (index) com banner e seções informativas
- Sistema de componentes modulares (header e footer)
- Arquivos base:
  - `index.html` - Página inicial
  - `header.html` e `headers.css` - Cabeçalho reutilizável
  - `footer.html` e `footer.css` - Rodapé reutilizável
  - `load_header.js` e `load_footer.js` - Scripts de carregamento
- Configuração do repositório GitHub
- README.md com documentação inicial
- CHANGELOG.md (este arquivo)

### Funcionalidades Iniciais
- Design responsivo
- Navegação entre páginas
- Componentes header/footer carregados via JavaScript
- Estrutura de pastas organizada
- CSS modular e reutilizável

## [0.1.0] - 2025-11-08

### Adicionado
- Configuração inicial do projeto
- Estrutura de pastas básica
- Commit inicial do repositório

## Tipos de Alterações

- `Adicionado` para novos recursos.
- `Modificado` para alterações em funcionalidades existentes.
- `Depreciado` para recursos que serão removidos em versões futuras.
- `Removido` para recursos removidos neste versão.
- `Corrigido` para qualquer correção de bugs.
- `Segurança` em caso de vulnerabilidades corrigidas.

## Política de Versionamento

- **MAJOR** (1.0.0): Mudanças incompatíveis com versões anteriores
- **MINOR** (1.1.0): Adição de funcionalidades compatíveis
- **PATCH** (1.0.1): Correções de bugs compatíveis
